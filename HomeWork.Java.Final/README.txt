System Requiremts:
Java RE: 1.8.0_241

Used Tools:
JDK: OpenJDK 13.0.1
IDE: Eclipse
OS: Windows 10 x64

Структура:
Каждое задание помещено в свой каталог. Исполняемый JAR-файл расположен в "bin" под-каталоге задания (Task1/bin/Lines.jar).
Для простоты запуска, рядом с исполняемым JAR-файлом лежит системный скрипт ("bat" для Windows, "sh" для Bash) для запуска (java -jr Lines.jar).
